## Description

NewPipe n'utilise aucune bibliothèque Google ni l'API YouTube. Les sites web sont uniquement analysés pour récupérer les informations nécessaires, pour que l'appli puisse être utilisée sur des appareils sans les services Google installés. De plus, pour utiliser NewPipe – un logiciel libre et « gauche d'auteur » _(copyleft)_  – vous n'avez pas besoin de compte YouTube. 

### Fonctionnalités

* Rechercher des vidéos
* Afficher des informations générales à propos des vidéos
* Regarder des vidéos YouTube
* Écouter des vidéos YouTube
* Lecteur flottant (aussi appelé lecteur popup)
* Chosir le lecteur permettant de regarder des vidéos
* Télécharger des vidéos
* Télécharger uniquement l'audio
* Ovurir une vidéo dans Kodi
* Afficher les vidéos suivantes / liées
* Chercher sur YouTube dans une langue spécifique
* Regarder / bloquer du contenu avec limite d'âge
* Afficher des informations générales à propos des chaines
* Rechercher des chaines
* Regarder les vidéos d'une chaine
* Support d'Orbot / Tor (pas encore directement)
* Support des résolutions 1080p / 2K / 4K
* Regarder l'historique
* S'abonner à des chaines
* Historique de lecture
* Voir / rechercher des listes de lectures
* Mettre des vidéos dans une file de lecture
* Créer des listes de lectures localement
* Sous-titres
* Support des flux en direct
* Afficher les commentaires

### Services supportés

NewPipe supporte plusieurs services. Notre [documentation](https://teamnewpipe.github.io/documentation) donne plus d'informations à propos de l'ajout d'un service dans l'extracteur et l'appli. Veuillez nous contacter si vous avez l'intention d'ajouter un nouveau service. Les services actuellement supportés sont :

* YouTube
* SoundCloud \[beta\]
* media.ccc.de \[beta\]
* PeerTube \[beta\]

## Mises à jour

Lorsque des modifications sont effectués au code de NewPipe (soit grâce à de nouvelles nouveautés, soit à cause de corrections de bugs), une mise à jour finira par être disponible. Le format de version est x.xx.x . Pour obtenir les nouvelles versions, vous pouvez :
 1. Compiler l'application et créer un fichier APK vous-même. C'est la méthode la plus rapide, mais est plus compliquée, par conséquent nous vous conseillons d'utiliser une des autres méthodes.
 2. Ajouter notre dépôt F-Droid et installer NewPipe depuis celui-ci. Les mises à jour sont disponibles dès qu'elles sont publiées par l'équipe NewPipe. Veuillez lire les instructions pour ajouter notre dépôt F-Droid : https://newpipe.schabi.org/FAQ/tutorials/install-add-fdroid-repo/
 3. Télécharger depuis un fichier APK depus les [publications GitHub (releases)](https://github.com/TeamNewPipe/NewPipe/releases) et installez l'appli. Les mises à jour seront disponibles dès qu'elles sont publiées par l'équipe NewPipe.
 4. Metre à jour via F-Droid. C'est la méthode la plus lente, puisque F-Droid doit être compiler et créer un fichier APK, le signer, puis publier la mise à jour à ses utilisateurs.

Nous recommendons la méthode 2 à la plupart de nos utilisateurs. Les fichiers APK installés via la méthode 2 ou 3 sont compatibles, mais pas avec la méthode 4. Nous utilisons notre signature pour signer les fichiers APK, mais F-Droid utilise leur propre signature. Android compare les signatures lors des mises à jour pour des raisons de sécurité, voilà pourquoi ces méthodes sont incompatibles.

Cependant, si vous voulez changer de source pour certaines raisons (par ex. une fonctionnalité principale de NewPipe ne fonctionne plus et F-Droid n'a pas encore publié la mise à jour), nous vous recommendons cette procédure :
1. Sauvegardez vos données via les Paramètres > Contenu -> Exporter la base de donnée, afin de conserver les abonnements, listes de lecture et l'historique
2. Désinstallez NewPipe
3. Téléchargez NewPipe depuis la nouvelle source et installer le
4. Importez vos données précédemment exportées via les Paramètres > Contenu > Importer la base de données

## Politique de confidentialité

Le projet NewPipe cherche a fournir une expérience anonyme et privée pour les services multimédias en ligne.
Par conséquent, notre appli ne collecte rien sans votre consentement. La poliique de confidentialité de NewPipe explique en détail quelles données sont envoyées et conservées lorsque vous signaler une erreur, ou commentez sur notre blog. Vous pouvez trouver ce document [ici](https://newpipe.net/legal/privacy/).
